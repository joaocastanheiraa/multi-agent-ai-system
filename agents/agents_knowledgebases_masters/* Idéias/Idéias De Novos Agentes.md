```
## Agente Que Cria Novos Agentes
    - Entende o problema que deve resolver
    - Entende qual o seu objetivo 
    - Entende o que ele terá de material para iniciar seus trabalhos
    - Pesquisa e descobre o que vai precisar descobrir e aprender a mais para conseguir entregar o melhor resultado possível a partir do que foi solicitado
    - Divide a entrega em pilares mais importantes que devem ser devidamente trabalhados de forma indivual e já entende como isso deve ser feito e por quem
    - Cria sub agentes especializados para cada um dos pilares desenhados (1 para cada pilar)
    - Cria os prompts a partir do template já existente
    - Revisa se todos os prompts estão os melhores possíveis, se estão devidamente conectados entre si assim como deveriam (imputs e outputs definidos de forma clara e com sequência bem definida), se estão usando as melhores técnicas de engenharia de prompt e rag.
```




```
## Agente Que Cria Novos Documentos Para As Bases De Conhecimento
    - Entende a estrutura de agentes e sub agentes e quais sao todos seus imputs, outputs esperados, objetivos, dificuldades
    - Pesquisa a partir disso como fazer algo da melhor forma possivel, descobre as técnicas, palavras chave e pilares que precisam ser seguidos
    - Cria lista de documentos que devem ser desenvolvidos
    - Manda sub agente criar
    - Manda outro sub agente otimizar
    - Manda outro otimizar colocando tecnicas de RAg e AI friendly
    - Manda o comando inserir o documento no lugar certo dentro da base de conhecimento
    - Sobe o documento chunknizado para o banco de dados vetorial
```
